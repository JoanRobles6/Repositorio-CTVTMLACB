En la carpeta Imagenes los conjuntos de graficas y matrices estan separados por carpetas de cada metodo de Vectorizacion con cada conjunto de Datos
Dentro de cada carpeta estan 15 imagenes, cada una esta titulada de acuerdo al modelo del que proviene

In the 'Imagenes' folder, the sets of graphs and matrices are separated by folders for each vectorization method within each data set.
